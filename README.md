# Python Restaurant Management System

## Overview
This project is a Python-based restaurant management system designed to simulate basic restaurant operations through a command-line interface.

## Features
- Add, remove, and update menu items
- View and manage inventory
- Process customer orders and checkout
- Calculate GST and discount automatically
- Handle simple online order processing
- Validate user input for menu options and quantities

## Technologies Used
- Python
- Lists and dictionaries
- Command-line interface

## Project Purpose
This project was developed to strengthen my practical Python programming skills by applying data structures, control logic, input validation, and billing calculations in a restaurant management scenario.

## Project Structure
- `restaurant_management_system.py`: main program file containing menu, inventory, order processing, and billing functions

## How to Run
1. Make sure Python is installed on your computer.
2. Download the project file.
3. Run the program using Python.
4. Follow the menu options shown in the terminal.
